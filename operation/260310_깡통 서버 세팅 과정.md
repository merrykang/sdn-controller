## 깡통 서버 세팅 과정 (파일 1개씩 만들어볼 것)

#### 1. 서버 OS 설치
- 구운 usb 로 하면 됨
- 처음 설정하는 username, password 는 완전 중요하다

---

#### 2. IP 세팅 ~ SSH 접속
- 전반적인 개념이랑 흐름 알 필요 있음

---

#### 3. 웹서버 설치
- 팀원들의 리눅스 계정 만들기 + sshd 컨피그 파일 수정(password 접속 YES)
- git clone 받기: ssh key 등록해서 받아야함
- jdk 설치: security-test/scripts/install/install_onos-jdk.sh
- 지도 서버 등록
- 키클록 등록: 일단 db 안 보는 버전으로 등록함. db 보는 버전으로 수정해야함
	- db 안 보는 버전 실행 파일들

---

#### 4. 로컬 컨트롤러 설치
- 서비스들 기본 파일, 인증서 없는 경우 존재: scp 한 후 인증서를 신뢰되는 저장소에 강제 등록 (~ fresh) 해서 해결

---

#### 5. 컴퓨팅 노드 설치 
- go 설치
- chromium 설치
- sphere 인증서 오류 해결
- panic 오류 해결 (포트 점유 문제 / nil 처리되지 않았던 것)
- systemctl start 서비스 할 때, too many files 되어서 실행되지 않았던 문제 해결
